🚪 SKUD Portal
Система контроля и управления доступом. Веб-приложение для управления правами доступа сотрудников.

📦 Установка и запуск
1. Клонировать репозиторий
git clone https://github.com/desolut1on/skud-portal.git

2. Перейти в папку проекта
cd skud-portal

3. Установить зависимости
npm install

4. Запустить проект
npm run dev

📁 Структура проекта
src/
├── contexts/ – контексты (авторизация)
├── pages/ – страницы приложения
├── styles/ – глобальные стили
├── assets/ – изображения
├── App.jsx – главный компонент
└── main.jsx – точка входа

👥 Роли пользователей
🔹 Администратор – полный доступ к системе
🔹 Оператор – управление доступом сотрудников
🔹 Сотрудник – просмотр своего статуса доступа
